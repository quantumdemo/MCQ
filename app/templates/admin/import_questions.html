CSV Format Instructions

Your CSV file must have a header row with the following columns:

question_text, subject, topic, difficulty, explanation, option1_text, option1_is_correct, option2_text, option2_is_correct, ...
  • question_text, subject, topic, difficulty: These fields are required.
  • explanation: This field is optional.
  • optionN_text, optionN_is_correct: Provide at least 2 and up to 5 pairs of option columns (e.g., option1_text to option5_text).
  • For optionN_is_correct columns, use 1 or TRUE for a correct answer, and 0, FALSE, or leave it empty for an incorrect answer.
